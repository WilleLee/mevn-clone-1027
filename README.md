client build -> dist 폴더 옮기기 -> server/dist -> server 실행

1. client 폴더 이동
2. npm ci
3. npm run build
4. ./client/dist -> ./server/dist 이동

5. server 폴더 이동
6. npm ci
7. babel
8. node app.js
